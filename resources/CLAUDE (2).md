# פרויקט WordMaster - אפליקציית Flashcards לאנגלית

## רקע על המשתמש
- לא מתכנת, יודע רק קצת Python
- צריך ללמוד אנגלית (כנראה לבחינה/מבחן)
- רוצה ללמוד בדרכים או ברגעים פנויים בעבודה - לכן צריך סטים קצרים
- יש לו PDF עם מילון מילים שצריך לחלץ ממנו (עדיין לא הועלה)

## מה כבר נבנה
קובץ `flashcards.jsx` - אפליקציית React מלאה הכוללת:

### 3 מצבי לימוד
1. **כרטיסיות (Flashcard)** - הפוך וזכור, ~3 דקות
2. **בחירה מרובה (Multiple Choice)** - 4 תשובות אפשריות, ~5 דקות
3. **AI מתקדם** - שאלות דינמיות שמשתנות לפי סוג:
   - השלמת משפטים
   - תרגום מאנגלית לעברית
   - תרגום מעברית לאנגלית
   - הבנה מהקשר (עם משפט לדוגמה)

### פיצ'רים שממומשים
- **Spaced Repetition** - מילים קשות חוזרות יותר (לפי מספר טעויות ורמה)
- **גודל סט מותאם** - 5/10/15/20 מילים לבחירה
- **מעקב התקדמות**:
  - סטטיסטיקות מפורטות
  - התפלגות רמות (חדש → שולט)
  - רשימת "מילים מאתגרות"
  - אחוז דיוק כללי
- **Streak** - מעקב ימים רצופים
- **שמירה ב-localStorage** - ההתקדמות נשמרת בין סשנים
- **תמיכה מלאה בעברית** - RTL, פונט Heebo
- **עיצוב מודרני** - צבעים כהים, גרדיאנטים, אנימציות

### מבנה הנתונים
```javascript
// מילה במאגר
{
  id: 1,
  english: "abundant",
  hebrew: "שופע, רב",
  example: "The region has abundant natural resources."
}

// סטטיסטיקות למילה
{
  correct: 5,      // מספר תשובות נכונות
  incorrect: 2,    // מספר טעויות
  lastSeen: timestamp,
  level: 3         // 0-5, משפיע על תדירות החזרה
}
```

## מה עוד צריך לעשות

### עדיפות גבוהה
1. **לחלץ מילים מה-PDF** - המשתמש יעלה PDF עם מילון, צריך לחלץ את המילים ולהוסיף למאגר
2. **לשפר את מצב AI** - כרגע השאלות הדינמיות בסיסיות, אפשר:
   - להשתמש ב-API של Claude ליצירת שאלות אמיתיות
   - להתאים שאלות למילים שהמשתמש טועה בהן
   - ליצור משפטי הקשר חדשים

### עדיפות בינונית
- להוסיף אפשרות לייבא/לייצא מילים
- להוסיף מצב "חזרה מהירה" רק על מילים בעייתיות
- להוסיף צלילים/הגייה

### הערות לפיתוח
- המשתמש לא מתכנת - לתת הסברים פשוטים
- לשמור על הקוד בקובץ אחד (single file React component)
- העיצוב חשוב - להמשיך עם הסגנון הקיים
- RTL חייב לעבוד טוב

## המשימה הבאה
**לעדכן את `flashcards.jsx` עם כל המילים מ-`vocabulary.md`**

הקובץ `vocabulary.md` מכיל את כל המילים מחולקות לקטגוריות:
- קישורים ומילות קישור (Connectors & Transitions)
- מילים כלליות (General Words)
- משפט ומשפטיות (Legal & Judicial)
- עושר ומעמד (Wealth & Status)
- עבודה וחברה (Work & Society)
- דיון ושכנוע (Discussion & Persuasion)
- חינוך ומשפט (Education & Law)
- פשע ומשפט (Crime & Justice)
- רגשות ואופי (Emotions & Character)
- מדע וטבע (Science & Nature)

כל מילה כוללת:
- המילה באנגלית
- תרגום לעברית
- משפט לדוגמה

## איך לעדכן
צריך להמיר את הטבלאות ב-vocabulary.md לפורמט JSON:
```javascript
{ id: 1, english: "adjacent to", hebrew: "סמוך ל-, צמוד ל-", example: "The park is adjacent to the school." }
```
